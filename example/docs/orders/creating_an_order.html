Orders API

Creating an order

Parameters

Name Description
name Name of order
paid If the order has been paid for
email Email of user that placed the order

Request

Headers

Accept: application/json
Host: example.org
Content-Type: application/x-www-form-urlencoded
Cookie:

POST /orders

order[name]=Order 1
order[paid]=true
order[email]=email@example.com
